# Option
This type defines the selection option that is shown when the user enters text

## FieldOption
A field option is shown when the text enters matches a field name.
### type
Always 'f'
### key
unique key
### field
Field to select

## SingleOption
A single selection option
### key
unique key
### field
Field to select
### type
Always 's'
### value
Option value
### Icon
Icon to show
### text
Option text

## RangeOption
A range selection option
### key
unique key
### field
Field to select
### type
Always 'r'
### value
Option value from
### text
From display text
### Icon
From icon
### valueTo
Option value to
### textTo
To display text
### IconTo
to Icon
### displayText
Text to disply for the option
